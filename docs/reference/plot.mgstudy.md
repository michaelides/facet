# Plot Method for mgstudy Objects

Plot Method for mgstudy Objects

## Usage

``` r
# S3 method for class 'mgstudy'
plot(x, type = c("variance", "proportion", "forest"), ...)
```

## Arguments

- x:

  An mgstudy object.

- type:

  Type of plot: "variance", "proportion", or "forest".

- ...:

  Additional arguments passed to plotting functions.

## Value

A ggplot object (invisibly).
