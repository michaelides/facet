# Format Sample Size Info for Display

Creates a formatted string representation of sample size information for
use in print and summary methods.

## Usage

``` r
format_sample_size_info(sample_size_info, indent = 1)
```

## Arguments

- sample_size_info:

  List from calculate_sample_size_info().

- indent:

  Number of spaces for indentation (default 1).

## Value

Character vector of formatted lines.
