# Extract Variance Draws from G-Study for Viable Recalculation

Extract Variance Draws from G-Study for Viable Recalculation

## Usage

``` r
extract_variance_draws_from_gstudy(gstudy_obj, n_draws = 1000)
```

## Arguments

- gstudy_obj:

  A gstudy object

- n_draws:

  Number of pseudo-draws to generate for mom backend
