# Check if Component is Part of Object Specification

Determines whether a variance component is part of the object of
measurement specification (which can include multiple components).

## Usage

``` r
is_object_component(component, object_spec)
```

## Arguments

- component:

  Character string naming the variance component.

- object_spec:

  Character vector of object component names.

## Value

TRUE if the component is part of the object specification.
