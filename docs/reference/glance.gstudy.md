# Glance Method for gstudy Objects

Returns a one-row summary of model statistics.

Returns a one-row summary of model statistics for multivariate
G-studies.

## Usage

``` r
# S3 method for class 'gstudy'
glance(x, ...)

# S3 method for class 'mgstudy'
glance(x, ...)
```

## Arguments

- x:

  An mgstudy object.

- ...:

  Additional arguments (ignored).

## Value

A one-row tibble.

A one-row tibble.
