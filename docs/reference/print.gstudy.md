# Print Method for gstudy Objects

Print Method for gstudy Objects

## Usage

``` r
# S3 method for class 'gstudy'
print(x, digits = 3, scale = c("variance", "sd"), ...)
```

## Arguments

- x:

  A gstudy object.

- digits:

  Number of digits to display.

- scale:

  Scale for displaying results: "variance" (default) or "sd".

- ...:

  Additional arguments (ignored).

## Value

Invisibly returns x.
