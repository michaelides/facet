# Example Dataset for Cronbach's Alpha

A simulated dataset containing responses for a simple person-by-item
design, suitable for demonstrating Cronbach's alpha and simple
G-studies.

## Usage

``` r
alpha_example
```

## Format

A data frame with 2000 rows and 4 variables:

- Employee_ID:

  Integer identifier for the employee

- i:

  Item identifier (e.g., "Item_1")

- y:

  Response value (numeric)

- p:

  Person identifier (e.g., "p 1")

## Source

Simulated data.
